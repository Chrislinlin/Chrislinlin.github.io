什麼是 JSON

JSON 是一種資料交換格式，是個以純文字為基底去儲存和傳送簡單結構資料。

  • 可以透過特定的格式去儲存任何資料(字串,數字,陣列,物件)
  • 透過物件或陣列來傳送較複雜的資料

一個 JSON 例子

以 JSON 描述我開的車，我們可以看到 JSON 檔是個以 key / value 所構成的檔案，在裡面 Value 可以為字串/ 數字/ 布林值/ 物件或是陣列

常用的方法

  • JSON.stringify() 將 array 轉成 string
  • JSON.parse() 將 string 轉成 array
