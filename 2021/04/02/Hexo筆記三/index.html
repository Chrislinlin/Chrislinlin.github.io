Hexo 筆記三

推上 github

  1. 先在 github 新增一個 repository，命名邏輯為

  1. 複製該 repository 的網址(例：git@github.com:Chrislinlin/Chrislinlin.github.io.git)
  2. 用 vscode 編譯器，打開_config.yml 檔案，修改deployment的部分
  1. 安裝 git 部署套件
  1. 輸入指令，將網站佈署到伺服器上

  1. 設定 repository 的主題
    在 github 的該 repository 頁面，進入「setting」

    往下滑，點按「change theme」

  2. 在瀏覽器中查看網址，成功的看到部落格推上去！https://Chrislinlin.github.io/

  3. 若有檔案更新，則重複輸入指令

更換主題

  1. 尋找
    前往hexo 主題官網尋找有興趣的主題
  2. 安裝
    • 前往該主題的 github 頁面，勇敢的 git clone 他，這裡示範安裝 icarus 主題
    • 在 vscode 開啟當初創建部落格資料，輸入指令 clone 想要安裝的主題
  1. 修改_config.yml
    原本內建的主題是landscape，這裡修改為 icarus
  2. 重新產生專案
  1. 完成
