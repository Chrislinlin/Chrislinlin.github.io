Javascript_觀念_Array_1_forEach()/map()

在 JS 中，陣列是個物件，沒有長度或型別規定。

  • 宣告 array 數組/陣列var car = ["toyata", "volvo", "BMW"];

  • 取得陣列值，透過索引值由 0 開始，上述例子 car[0]=>toyota

  • 改變原始陣列，有許多屬性/方法

    • 如 push()、pop()、shift()、unshift()、reverse()、slice(a,b,c)、sort()排序….

方法

forEach()

  • 把我們想要執行的函式，從頭到尾逐一（iterates through）的把陣列裡的元素帶進去這個函式處理一遍

  • 語法

    arr.forEach(function callback(currentValue[, index[, array]]) {

    //your iterator

    }[, thisArg]);

  • callback 函式必須，其餘兩個是 optional

取得加總數

印出帶有索引值，多了第二個參數

把整個陣列列印，多了第三個參數

他會根據你的陣列個數列印幾多少次的整個陣列

統計出現次數

這裡有一個觀念，要是陣列裡待有純字串的數字或是字串，當我們在使用 forEach 時，要是他本身沒有在原陣列被宣告，那字串及指定的值會變成key:value的方式呈現。

map()

將原始的變數運算後重新組合一個 ==新的陣列==，記得一定要重新指派給新的變數，不然原本的陣列是不會被改變的。

  • 語法

    let new_array = arr.map(function callback( currentValue[, index[, array]]) {

    // return element for new_array

    }[, thisArg])

  • 沒有重新指派
  • 對每個陣列元素加工
  • 把 String 陣列轉為數字陣列
  • 對陣列元素校正
