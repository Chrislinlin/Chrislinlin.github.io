Javascript_觀念_函式及作用域

函式是一個指令或多個指令一起被包在括號裡面，可以重複被調用的==物件==。

定義方式

函式宣告

  • 表示式：JS 可以指派 function 給一個變數（其他語言無法）

  • 例子一

  • 直接式：直接定義函式

  • 例子二

函式運算

以剛剛例子二範例，我們函式 add，會回傳參數 a 與參數 b 的相加結果，這個就是內的運算

函式可以做的事

函式是一個物件，物件可以做的事，函式都可以做。

  • 可以當作參數被傳入
  • 可以有自己的屬性
  • 可以被呼叫
  • 可以沒有名字

作用域 Scope

函式內所定義的變數只屬於==函式內==使用。

同樣的變數在全域裡 x 是 100，但是在函式 hi 裡，x 有被指派新的值，但這也僅限在函式內部使用，在外面 x 還是全域變數 100。
