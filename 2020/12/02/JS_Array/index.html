JS_Array

JS_Array

在 JS 中，陣列是個物件，沒有長度或型別規定。

  • 宣告 array 數組/陣列var car = ["toyata", "volvo", "BMW"];

  • 取得陣列值，透過索引值由 0 開始，上述例子 car[0]=>toyota

  • 改變原始陣列，有許多屬性/方法

    • 如 push()、pop()、shift()、unshift()、reverse()、slice(a,b,c)、sort()排序….

方法

  1. forEach()

  • 把我們想要執行的函式，從頭到尾逐一（iterates through）的把陣列裡的元素帶進去這個函式處理一遍

  • 語法： arr.forEach(callback, thisArg);

  • 回傳值： undefined。

  • 參數： callback 函式與 thisArg

  1. splice(a, b, c)可以移除或新增陣列的元素，它包含了三個參數，
  • a 是要移除或要添加的序列號碼 ( 必填 )，
  • b 是要移除的長度 ( 選填，若不填則第一個號碼位置後方的所有元素都會被移除，若設定為 0 則不會有元素被移除 )，
  • c 是要添加的內容 ( 選填 )
  1. toString()/ join()方法陣列轉換成字符串
  2. pop()/ push() 把最後一個位置元素取出/新增。
  1. slice()方法為在陣列中某個片段切出新陣列，可以指定位置（索引）

  1. spread syntax …將剩餘的…轉換給…array - 將 array array1 的內容展開給 array2(可是字串/ 函式參數/物件 )
    .js= var array1 = [1, 2, 3, 4] var array2 = [5, 6, 7, 8]; array2.forEach( num => array1.push(num) ) console.log( array1 ) // [ 1, 2, 3, 4, 5, 6, 7, 8 ]
    參考筆記

  2. map()

將原始的變數運算後重新組合一個新的陣列。

  • 對每個陣列元素加工
  • 對陣列元素校正
  1. reduce()
  • reduce() 進行數值加總
  1. filter()
  • 會回傳一個陣列，條件是 return 後面必須為 true
  1. find()
  • 和 filter 很像都會回傳一個陣列，條件是 return 後面條件為 true，但只會回傳第一個條件為 true 的值
